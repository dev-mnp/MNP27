# User Guide

## Purpose

User Guide explains how to use the app module by module.

## How to use

- Choose a module from the left guide list.
- Read the End User section for workflow steps.
- Developers can read the Developer section for implementation notes.
- Edit the Markdown files when instructions need to be updated.

## System behavior

- Guide content is loaded from Markdown files.
- End-user files are stored in `core/user_guide/content/enduser`.
- Developer files are stored in `core/user_guide/content/developer`.
