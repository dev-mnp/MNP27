# Audit Logs

## Purpose

Audit Logs help trace who changed what and when.

## How to use

- Open Audit Logs when a value looks different from expected.
- Search or filter by user, action, module, or date where available.
- Use the log to understand the change history before correcting data.

## System behavior

- Audit Logs are for review, not editing.
- They support accountability and troubleshooting.
- If a record changed unexpectedly, check Audit Logs before making another correction.
