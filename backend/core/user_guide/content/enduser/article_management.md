# Article Management

## Purpose

Article Management maintains the master list of Aid, Article, and Project items.

## How to use

- Use Create to add a new item.
- Use Edit to update article name, token name, price, category, master category, active state, or combo state.
- Use Delete only when the item is no longer needed and safe to remove.
- Use filters and search to find items quickly.
- Use Export to download the article master.

## Price rule

- Price 0 means the item price stays editable in Application Entry.
- Price above 0 means the item is treated as fixed price.
- If changing a price affects saved application rows, the app shows a confirmation before updating matching rows.

## Duplicate protection

- Duplicate article names are warned.
- Duplicate token names are also warned.
- Article name, token name, category, and master category fields show existing values while still allowing new values.
