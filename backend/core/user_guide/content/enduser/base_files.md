# Base Files

## Purpose

Base Files store yearly reference data used by the rest of the app.

## What to upload

- District master file.
- Article list file.
- Past beneficiary history file.

## How to use

- Upload base files before Application Entry starts.
- Replace a base file only when the replacement is confirmed.
- After changing base files, review any dependent module that uses that data.

## System behavior

- District master supports district application creation.
- Article list supports Article Management and Application Entry item selection.
- Past history supports public Aadhaar verification.
