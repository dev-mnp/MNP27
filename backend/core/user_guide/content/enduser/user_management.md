# User Management

## Purpose

User Management controls who can access each module and what each user can do.

## How to use

- Create users for team members.
- Assign role and module permissions.
- Enable or disable access to menu modules.
- Reset passwords when required.
- Use dashboard page permissions when a user should or should not see extra dashboard pages.

## System behavior

- The left menu follows module permissions.
- A user may see only the modules assigned to them.
- If a screen is missing for a user, check User Management first.
