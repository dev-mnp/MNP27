# Vendors

## Purpose

Vendors stores supplier information used in purchase and ordering workflows.

## How to use

- Create vendor records with name and contact details.
- Edit vendor details when corrected information is available.
- Use search or filters to find vendors.
- Delete only when the vendor is no longer required and safe to remove.

## System behavior

- Vendor records help avoid repeated typing in purchase workflows.
- Keep vendor names consistent so purchase documents remain clean.
