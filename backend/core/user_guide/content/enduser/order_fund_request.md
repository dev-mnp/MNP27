# Order & Fund Request

## Purpose

Order & Fund Request is used to raise Aid requests and Article requests.

## Aid request

- Select Aid as the request type.
- Select the aid name.
- Use Quick Populate to add eligible beneficiaries.
- Save as Draft while checking details.
- Submit only after final review.

## Article request

- Select Article as the request type.
- Group article rows under the correct vendor.
- Review quantity and vendor details.
- Submit only when the request is final.

## Aadhaar duplicate check

- Before submitting Aid requests, Aadhaar numbers in the draft are checked against submitted fund requests.
- If a duplicate is found, the row shows which fund request already contains the Aadhaar.
- Public records marked Aadhaar not available show verification pending and block save or submit until fixed.

## Submit behavior

- Submit is final for this workflow.
- The app shows a confirmation because submitted requests affect downstream ordered quantities.
