# Dashboard

## Purpose

Dashboard gives a quick health view of the event data. Use it to check totals before opening detailed modules.

## How to use

- Open Dashboard from the left menu.
- Review application counts for District, Public, Institutions, and Others.
- Check total accrued value, planning total, district contribution, and underutilised district value.
- Use the quantity and value tree page after Seat Allocation to see how totals split by beneficiary type, item type, Waiting Hall quantity, and Token quantity.

## System behavior

- Dashboard does not create or edit records.
- If a number looks wrong, fix the source module instead of changing the dashboard.
- Access to dashboard pages can be controlled from User Management.
