# Purchase Order

## Purpose

Purchase Order prepares vendor-facing purchase documents.

## How to use

- Create a purchase order.
- Select or enter vendor details.
- Add item rows with quantity, unit price, line total, supplier article name, and description.
- Save as Draft while preparing.
- Submit when final.
- Download the purchase order PDF.

## System behavior

- Purchase Order is separate from Fund Request.
- Vendor information comes from Vendor Management where available.
- Submitted purchase orders are treated as finalized documents.
