# Labels & Tags

## Purpose

Labels & Tags generate printable labels from token-generated data.

## How to use

- Use Sync Data to pull the latest token-generated data.
- Preview a label group before downloading.
- Download the PDF only after the preview looks correct.
- Use article labels for physical items.
- Use beneficiary labels for District, Public, Institution, and Others badges or tags.

## System behavior

- Labels use token-generated data, not raw application data.
- Excluded token rows should not appear in labels.
- Others labels are separate from Institution labels.
