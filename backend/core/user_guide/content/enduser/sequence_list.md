# Sequence List

## Purpose

Sequence List decides the order in which items are handled for token generation and distribution.

## How to use

- Use Sync Data to copy data from Seat Allocation.
- Select unassigned items and move them into Sequenced Items.
- Sort or manually adjust the sequence.
- Save when the order is ready.
- Export the sequence output if needed.

## System behavior

- Sequence List works on unique items, not one beneficiary at a time.
- Items from Others are included if they exist in the synced allocation data.
- The saved sequence becomes the source for Token Generation.
